# Plan: 버튼클릭카운터3

## 메타데이터
- 기능명: 버튼클릭카운터3
- 날짜: 2026-02-20
- 레벨: Dynamic
- 우선순위: Medium

## 개요
버튼클릭카운터2에서 진화한 버전으로, **LocalStorage 영구 저장**과 **클릭 히스토리 로그** 기능을 추가한다.
페이지를 새로고침해도 카운터 값이 유지되며, 각 카운터의 변경 이력을 확인할 수 있다.

## 기능 요구사항

### 핵심 기능
1. **카운터 기본 기능** (버튼클릭카운터2 기반)
   - 다중 카운터 (추가/삭제)
   - 이름 편집
   - 증감 단위 선택 (1, 5, 10)
   - 합계 표시

2. **LocalStorage 지속성** (신규)
   - 카운터 값, 이름, 단위를 localStorage에 자동 저장
   - 페이지 새로고침 시 마지막 상태 복원
   - "초기화" 버튼으로 전체 리셋

3. **클릭 히스토리** (신규)
   - 각 증가/감소 버튼 클릭 시 로그 기록
   - 히스토리 형식: `[시간] 카운터명 +1 → 5`
   - 최근 10개 항목만 표시
   - 히스토리 접기/펼치기 토글

## 기술 스택
- Next.js App Router (Client Component)
- TypeScript
- Tailwind CSS
- localStorage API (Web Storage)

## URL 경로
- `/counter3`

## 성공 기준
- LocalStorage에 저장되어 새로고침 후에도 값 유지
- 히스토리 로그가 최근 10개 표시
- 빌드 에러 없음
- 기존 counter2 기능 모두 포함
