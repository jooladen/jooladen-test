# Design: 버튼클릭카운터2

## 컴포넌트 구조

```
src/app/counter2/
└── page.tsx          # 메인 페이지 (use client)
```

## 데이터 모델

```typescript
interface Counter {
  id: number;
  name: string;
  value: number;
}

// State
const [counters, setCounters] = useState<Counter[]>([{ id: 1, name: "카운터 1", value: 0 }]);
const [step, setStep] = useState<number>(1);
```

## UI 레이아웃

```
┌─────────────────────────────────────────┐
│         버튼 클릭 카운터 2               │
│  증감 단위: [+1] [+5] [+10]             │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 카운터 1         [이름수정]  [X] │   │
│  │      42                         │   │
│  │  [-step]        [+step]         │   │
│  └─────────────────────────────────┘   │
│  ┌─────────────────────────────────┐   │
│  │ 카운터 2         [이름수정]  [X] │   │
│  │       8                         │   │
│  │  [-step]        [+step]         │   │
│  └─────────────────────────────────┘   │
│                                         │
│  합계: 50          [카운터 추가]        │
└─────────────────────────────────────────┘
```

## 기능 명세

### 증감 단위 선택
- step 값: 1, 5, 10
- 선택된 단위 버튼 활성화 스타일

### 카운터 카드
- 이름 인라인 편집 (클릭 시 input으로 전환)
- +step / -step 버튼 (값이 0 미만 허용 안 함)
- X 버튼으로 삭제 (최소 1개 유지)

### 합계
- 모든 카운터 value 합산 표시
- 실시간 업데이트

### 카운터 추가
- 클릭 시 새 카운터 추가 (기본값 0, 이름 "카운터 N")

## 색상 테마
- 배경: bg-gradient-to-br from-green-50 to-teal-50
- 카운터 값: text-green-600
- 증가 버튼: bg-green-500
- 감소 버튼: bg-red-400
- 추가 버튼: bg-teal-500
