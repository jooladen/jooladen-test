# Plan: 버튼클릭카운터

## 개요
- 기능명: 버튼클릭카운터
- 날짜: 2026-02-20
- 레벨: Dynamic (Next.js + TypeScript + Tailwind CSS)

## 목표
버튼을 클릭할 때마다 숫자가 증가하는 인터랙티브 카운터 구현

## 요구사항
1. 현재 클릭 횟수를 화면에 표시
2. "클릭!" 버튼을 클릭하면 카운트 +1 증가
3. "초기화" 버튼을 클릭하면 카운트 0으로 리셋
4. Tailwind CSS를 활용한 스타일링
5. TypeScript 타입 안전성 보장

## 성공 기준
- [ ] 초기 카운트 0 표시
- [ ] "클릭!" 버튼 클릭시 카운트 증가
- [ ] "초기화" 버튼 클릭시 카운트 0 리셋
- [ ] Tailwind CSS 스타일 적용 (카드 형태, 버튼 스타일)
- [ ] TypeScript 에러 없음
- [ ] 빌드 성공

## 구현 범위
- src/app/page.tsx: 카운터 UI (Client Component)
- useState 훅으로 상태 관리

## 제외 범위
- 백엔드 저장 없음 (페이지 새로고침시 리셋)
- 인증 기능 없음
