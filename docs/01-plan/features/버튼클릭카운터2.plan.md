# Plan: 버튼클릭카운터2

## 개요
기존 버튼클릭카운터의 고급 버전. 단순 카운터에서 다양한 기능이 추가된 인터랙티브 카운터.

## 목표
- `/counter2` 경로에 배치
- 기존 카운터와 차별화된 고급 기능 제공

## 핵심 기능
1. **다중 카운터**: 여러 개의 카운터를 독립적으로 관리
2. **증감 단위 설정**: +1, +5, +10 단위로 클릭 가능
3. **카운터 이름 지정**: 각 카운터에 이름 설정 가능
4. **카운터 삭제**: 불필요한 카운터 삭제
5. **합계 표시**: 모든 카운터 합계 실시간 표시

## 기술 스택
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- React useState (클라이언트 컴포넌트)

## 경로
- `/counter2` → `src/app/counter2/page.tsx`

## 완료 기준
- 카운터 추가/삭제 동작
- 증감 단위 변경 동작
- 합계 실시간 계산
- 빌드 오류 없음
