# Design: 버튼클릭카운터

## 컴포넌트 설계

### 페이지: src/app/page.tsx
```typescript
"use client";

import { useState } from "react";

export default function Home() {
  const [count, setCount] = useState<number>(0);

  return (
    <main className="min-h-screen flex items-center justify-center bg-gray-100">
      <div className="bg-white rounded-2xl shadow-lg p-10 flex flex-col items-center gap-6">
        <h1 className="text-2xl font-bold text-gray-700">버튼 클릭 카운터</h1>
        <p className="text-6xl font-extrabold text-blue-600">{count}</p>
        <div className="flex gap-4">
          <button
            onClick={() => setCount(count + 1)}
            className="px-6 py-3 bg-blue-500 text-white font-semibold rounded-xl hover:bg-blue-600 active:scale-95 transition-all"
          >
            클릭!
          </button>
          <button
            onClick={() => setCount(0)}
            className="px-6 py-3 bg-gray-200 text-gray-700 font-semibold rounded-xl hover:bg-gray-300 active:scale-95 transition-all"
          >
            초기화
          </button>
        </div>
      </div>
    </main>
  );
}
```

## UI 설계
- 배경: 연한 회색 (bg-gray-100)
- 카드: 흰색 + 그림자 + 둥근 모서리 (rounded-2xl shadow-lg)
- 카운트 표시: 6xl 굵은 파란색 텍스트
- 클릭 버튼: 파란색 배경, hover 효과, 클릭 축소 애니메이션
- 초기화 버튼: 회색 배경, hover 효과

## 파일 구조
```
src/
  app/
    page.tsx    ← 수정 대상 (Client Component)
    layout.tsx  ← 변경 없음
    globals.css ← 변경 없음
```

## 기술 결정
- Client Component 사용 ("use client" 지시어)
- useState로 카운트 상태 관리
- 별도 컴포넌트 분리 없음 (단순 기능)
- TypeScript 제네릭 타입 명시 (useState<number>)
