# Design: 헬로월드

## 컴포넌트 설계

### 페이지: src/app/page.tsx
```typescript
// Server Component (기본)
export default function Home() {
  return (
    <main className="min-h-screen flex items-center justify-center bg-white">
      <h1 className="text-4xl font-bold text-gray-800">Hello, World!</h1>
    </main>
  )
}
```

## UI 설계
- 레이아웃: 화면 전체 가운데 정렬 (flex + items-center + justify-center)
- 배경: 흰색 (bg-white)
- 텍스트: 4xl 굵은 폰트, 회색 (text-gray-800)
- 반응형: min-h-screen으로 화면 높이 전체 사용

## 파일 구조
```
src/
  app/
    page.tsx    ← 수정 대상
    layout.tsx  ← 변경 없음
    globals.css ← 변경 없음
```

## 기술 결정
- Server Component 사용 (상태 관리 불필요)
- Tailwind CSS 유틸리티 클래스 활용
- 별도 컴포넌트 분리 없음 (단순 기능)
